# 🚀 Project Management SaaS

A modern full-stack Project Management SaaS application built to help teams manage projects, tasks, subtasks, users, notifications, insights, and schedules efficiently.

---

# ✨ Features

- 🔐 Authentication with Google Sign-In
- 📁 Project Management System
- ✅ Task & Subtask Management
- 👥 User Management
- 🔔 Real-Time Notification UI
- 📊 Analytics & Insights Dashboard
- 📅 Calendar & Scheduling System
- 🎨 Modern Responsive UI
- 🌙 Clean SaaS-style Design

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- DaisyUI
- Google Authentication

## Backend
- Node.js
- Express.js
- JWT Authentication

## Database
- MongoDB
